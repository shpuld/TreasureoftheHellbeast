# Treasure of the Hellbeast

A Ludum Dare 32 compo entry made in 48 hours by myself. Powered by the FTE QuakeWorld engine.
In this game you crawl a multi-floor dungeon, beating every monster up with your creative weapons.

Video: https://youtu.be/_sLmhtOGEy0

In order to play, you will also need the FTE QuakeWorld executable. http://fte.triptohell.info/
Place the executable in the same directory with default.fmf, and it should then launch the game.
